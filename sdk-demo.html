📦 How to Use

Option 1: Import in JavaScript/TypeScript

import { StellarTransactionVisualizer } from './dist/sdk/index.js'; const sdk = new StellarTransactionVisualizer({ isTestnet: true }); const tx = await sdk.getTransactionDetails('TX_HASH'); console.log(tx);

Option 2: Import Functions Directly

import { fetchTransaction, fetchContractTransactions, setNetwork } from './dist/services/stellar.js'; setNetwork({ isTestnet: true }); const tx = await fetchTransaction('TX_HASH');
